## 🔹 3. 변수와 상수

### 변수

```js

```

- `let`: 재선언 ❌ / 값 변경 ⭕
- `var`: 재선언 ⭕

### 상수

```js

```

- 값 변경 ❌

## 📦 4. 자료형

### 숫자

```js
console.log(1 / 0); // Infinity
console.log(1 * "hello"); // NaN
```

### 문자열

```js
let name = "정환";
let text = `hello ${name}`;
```

### Boolean / null / undefined

```js
let a = true;
let b = null;
let c;
```

## 🔄 5. 형 변환

### 묵시적 형 변환

```js
10 + "20"; // "1020"
```

### 명시적 형 변환

```js
Number("10");
parseInt("10개", 10);
String(2022);
Boolean(0); // false
```

## 🔀 7. 조건문

```js

  console.log("크다");
```

```js

    console.log("사과");

}
```

```js
}
```


### 콜백 함수

```js

}
```
